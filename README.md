### Featured Project

<table>
  <tr>
    <td width="58%">
      <h3>EnvDoctor</h3>
      <p>
        A framework-aware environment-variable diagnostic tool for modern JavaScript and TypeScript repos.
        It scans code, <code>.env*</code> files, monorepos, Turborepo config, GitHub Actions workflows, Vite, Next.js,
        and Node <code>--env-file</code> usage to explain why an env var is missing, leaked, stale, or misconfigured.
      </p>
      <p>
        <a href="https://github.com/Markussiin/envdoctor"><strong>View repository</strong></a>
      </p>
    </td>
    <td width="42%">
      <pre><code>npx envdoctor doctor

CRITICAL  DATABASE_URL is used but not documented
HIGH      VITE_STRIPE_SECRET_KEY may expose a secret
MEDIUM    Vite config reads process.env.APP_PORT</code></pre>
    </td>
  </tr>
</table>
